## Component Responsibilities

### Browser Extension
- **Input:** User actions (right-click, button click)
- **Output:** Visual feedback (notifications, dashboard)
- **Functions:**
  - Intercept downloads
  - Send files to backend
  - Display results
  - Track history
  - Show statistics

### FastAPI Backend
- **Input:** HTTP requests with files/paths
- **Output:** JSON responses with predictions
- **Functions:**
  - Load ML model
  - Handle file uploads
  - Extract features
  - Run predictions
  - Query VirusTotal
  - Track performance

### ML Model
- **Input:** Feature vector (72 dimensions)
- **Output:** Prediction (0=malicious, 1=benign) + confidence
- **Functions:**
  - Feature engineering
  - Classification
  - Probability estimation

### VirusTotal Integration
- **Input:** File hash (SHA-256)
- **Output:** Threat intelligence (family, detections)
- **Functions:**
  - Hash calculation
  - API querying
  - Rate limiting
  - Response parsing
